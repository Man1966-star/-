Πακέτο χωρίς macro (λειτουργεί σε πολλά περιβάλλοντα όπου ο άμεσος σύνδεσμος σε HTML αποτυγχάνει):

Περιεχόμενα:
- PAROUSIASI_link_to_bat.pptx  → κουμπί που καλεί το OpenGame.bat
- OpenGame.bat                  → ανοίγει το game_numbers_1-10_x_10-90.html στον προεπιλεγμένο browser
- game_numbers_1-10_x_10-90.html

Οδηγίες:
1) Κάνε Extract όλα τα αρχεία στον ίδιο φάκελο.
2) Άνοιξε το PAROUSIASI_link_to_bat.pptx και πάτα Enable Editing εάν ζητηθεί.
3) Άνοιξε Slide Show και πάτα το κουμπί.
4) Αν τα Windows δείξουν προειδοποίηση για .bat, διάλεξε 'More info' → 'Run anyway' (μόνο αν εμπιστεύεσαι το αρχείο).
